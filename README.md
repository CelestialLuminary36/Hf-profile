## 技术栈

- **React**：用于构建 UI。
- **ShadCN UI**：美观且易用的 UI 组件库。
- **Vite**：快速构建和开发工具，提供优异的开发体验。
- **React Router**：用于页面路由管理。
- **Tailwind CSS**：快速构建自定义设计。
- **TypeScript**：为 React 应用提供类型检查和更好的开发体验。

## 安装与运行

### 1. 克隆仓库

```bash
git clone https://github.com/CelestialLuminary36/Hf-profile.git
cd Hf-profile
```
### 2. 安装依赖
```bash
pnpm install
```

### 3. 运行
```bash
pnpm run dev
```
